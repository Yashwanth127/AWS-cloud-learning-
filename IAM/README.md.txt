AWS CLOUD LEARNING - DAY 1

What is AWS?
AWS (Amazon Web Services) is a cloud platform that provides services like storage, servers, databases, and more over the internet.

What is IAM?
IAM stands for Identity and Access Management.
It is used to control:
Who can access AWS
What they can do inside AWS

Key Concepts:

1. Root User
Main account of AWS
Has full access
Should not be used for daily tasks

2. MFA (Multi-Factor Authentication)
Extra security layer (OTP)
Protects AWS account from hackers

3. IAM Users
Individual users (like developers or team members)
Each user has login credentials

4. IAM Roles
- Temporary access without password
- Used by AWS services (like EC2)

5. Policies
- Permission rules written in JSON
- Example:
  - Allow access to S3
  - Deny deleting resources

6. IAM Dashboard
- Main page showing users, roles, policies
- Shows security recommendations
